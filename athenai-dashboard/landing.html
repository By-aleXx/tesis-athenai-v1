¿Qué es AthenAI?

AthenAI es un sistema de detección de intrusiones web (IDS) que combina múltiples modelos de machine learning en una arquitectura de defensa en profundidad. Analiza cada petición HTTP en tiempo real para identificar ataques como SQLi, XSS y anomalías de comportamiento.

🛡️

Detección Multicapa

Cuatro capas de análisis independientes eliminan falsos negativos mediante votación en cascada.

⚡

Inferencia Asíncrona

Arquitectura no bloqueante reduce la latencia un 78%: de 3,700 ms a 816 ms por request.

☁️

Cloud-Native AWS

Desplegado sobre Lambda, DynamoDB, S3 y CloudWatch con alta disponibilidad y escalado automático.

🤖

ML Híbrido

XGBoost + TF-IDF para detección rápida. DistilBERT + Isolation Forest para análisis semántico profundo.

Diseño del sistema

Arquitectura de 4 Capas

Defensa en profundidad: cada capa filtra amenazas con mayor precisión y mayor costo computacional. Las peticiones sospechosas escalan al siguiente nivel de análisis.

CAPA 0 Reglas estáticas
🔧

Nginx + ModSecurity

WAF basado en reglas OWASP CRS. Primera línea de defensa contra ataques conocidos. Latencia adicional: <1 ms.

CAPA 1 ML Supervisado
📊

XGBoost + TF-IDF

Clasificador supervisado con vectorización TF-IDF. Accuracy: 99.96%. F1: 99.97%. Cero falsos negativos en test.

CAPA 2 Ensemble
🌲

Random Forest

Ensemble de 100 árboles de decisión para segunda validación. F1: 99.95%. AUC-ROC: 99.9999%. Segunda opinión independiente.

CAPA 3 Deep Learning
🧠

DistilBERT + Isolation Forest

Transformer fine-tuned para análisis semántico + detección de anomalías no supervisada. Recall: 93.33%.

Cada petición HTTP atraviesa las capas de izquierda a derecha. Las amenazas detectadas se bloquean y registran en tiempo real.

Proponemos un sistema IDS multicapa que integra Nginx/ModSecurity, XGBoost+TF-IDF, Random Forest y DistilBERT con Isolation Forest en una arquitectura asíncrona cloud-native.
